# AI-Powered Student Career and Internship Assistant

## Overview

The **AI-Powered Student Career and Internship Assistant** is a web-based application designed to help students in Software Engineering, Computer Science, Information Technology, and other IT-related fields prepare for internships and early-career opportunities.

The system allows students to create a profile, list their technical skills, provide resume or career-goal information, and receive AI-assisted recommendations on career paths, missing skills, project ideas, learning plans, and internship readiness.

This project is being developed as a six-week software engineering internship project. It combines cloud engineering, artificial intelligence, cybersecurity, and practical software engineering skills.

## Problem Statement

Many undergraduate students in Software Engineering, Computer Science, Information Technology, and related IT fields study important classroom concepts such as programming, databases, networking, software design, and systems development. However, many still struggle to convert this knowledge into employable technical skills.

Students preparing for internships often lack structured guidance on which skills they are missing, which projects they should build, how to improve their resumes, and how to present themselves professionally to internship supervisors or employers.

This project addresses that gap by providing an AI-powered assistant that helps students assess their readiness, identify skill gaps, and receive practical recommendations for career growth and internship preparation.

## Target Users

The system targets undergraduate students in:

- Software Engineering
- Computer Science
- Information Technology
- Information Systems
- Cybersecurity
- Data Science
- Computer Engineering
- Other IT-related fields

Secondary users may include academic mentors, lecturers, industry mentors, career advisors, and internship supervisors who want to support students in preparing for practical computing careers.

## Core Features

The proposed system will include:

- Student profile creation
- Skills listing and self-assessment
- Resume or career-goal input
- AI-assisted career-path recommendation
- AI-assisted skill-gap analysis
- Project recommendation based on student interests
- Learning-path suggestion
- Recommendation history
- Secure handling of user data and API keys
- AWS-based cloud deployment

## Research Questions

This project is guided by the following research questions:

1. What career preparation challenges do students in Software Engineering, Computer Science, Information Technology, and related IT fields face when preparing for internships?
2. How can student profile data, skill self-assessment, and resume content be used to generate meaningful internship-readiness feedback?
3. What AI-powered feature provides the most practical value within a six-week internship project?
4. How can the system provide useful recommendations without becoming too broad or difficult to implement?
5. Which technology stack is most suitable for developing a secure, deployable, and maintainable minimum viable product?
6. How can AWS be used to demonstrate cloud engineering skills through deployment, database hosting, environment configuration, and monitoring?
7. What cybersecurity and privacy risks arise when handling student profiles, resume content, AI API requests, and authentication data?
8. How can the final system demonstrate software engineering competence through GitHub commits, API design, documentation, testing, deployment, and presentation?

## Proposed Technology Stack

### Frontend

- React with Vite
- HTML, CSS, and JavaScript
- Responsive user interface for profile input, skills input, resume input, and recommendation results

### Backend

The backend framework will be selected after Week 2 technical comparison.

Possible options:

- Django REST Framework
- FastAPI
- Node.js with Express

### Database

Possible options:

- PostgreSQL on Amazon RDS for production
- SQLite for early local testing

### AI Layer

Possible AI integration options:

- OpenAI API
- Google Gemini API
- Anthropic API
- Other approved AI API provider

The AI layer will be used for career guidance, skill-gap analysis, resume feedback, project recommendation, or learning-plan generation.

### Cloud Platform

The selected cloud platform is:

- Amazon Web Services

Possible AWS services:

- Amazon EC2 for backend hosting
- Amazon RDS for managed database hosting
- Amazon S3 for static files or future resume storage
- AWS IAM for access control
- Amazon CloudWatch for logs and monitoring

## Why AWS Was Selected

AWS was selected because it provides practical exposure to real cloud engineering concepts such as compute services, databases, storage, identity and access management, monitoring, and secure deployment.

Using AWS also strengthens the learning value of the internship because it allows the project to demonstrate:

- Cloud deployment
- Environment variable management
- Database hosting
- Security groups and access control
- Monitoring and logging
- Secure handling of credentials
- Portfolio-ready cloud experience
